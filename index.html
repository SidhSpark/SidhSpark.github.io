Phase 3: The Time Machine & Safety Net අදියර 3: කාල යන්ත්‍රය සහ ආරක්ෂක ජාලය

The Time Machine & Cloud Safe-House: Git & GitHub කාල යන්ත්‍රය සහ ඔන්ලයින් සුරක්ෂිතාගාරය: Git & GitHub

What if something goes wrong? What if your computer crashes? Git is our time machine. GitHub is our cloud safe-house, accessible from anywhere. Install Git. Create a GitHub account. වැඩ කරගෙන යද්දී යමක් වැරදුනොත්? පරිගණකයට යමක් වුනොත්? Git යනු අපේ කාල යන්ත්‍රයයි. GitHub යනු අපේ නිර්මාණ ලෝකයේ ඕනෑම තැනක සිට ප්‍රවේශ විය හැකි, වලාකුළු මත පදනම් වූ සුරක්ෂිතාගාරයයි (cloud safe-house). Git install කරන්න. GitHub ගිණුමක් සාදන්න.

Sharing with the World: Pushing to GitHub ලෝකය සමඟ බෙදාගැනීම: Pushing to GitHub

Now, let's send your local creation to your secure online safe-house. දැන් ඔබගේ දේශීය (local) නිර්මාණය, ලෝකයට පෙනෙන ඔන්ලයින් සුරක්ෂිතාගාරයට යවමු.

  1. On GitHub, create a "New repository" and name it `my-digital-footprint` (make sure it's Public). GitHub හි, "New repository" සාදා, එයට `my-digital-footprint` ලෙස නම දෙන්න (Public ලෙස තෝරන්න).
  2. GitHub will give you instructions. Copy the two commands that look like this and run them in your terminal: GitHub මගින් ලබා දෙන උපදෙස් වලින්, පහත විධාන දෙක පමණක් copy කර, ඔබගේ terminal එකේ run කරන්න: 
git remote add origin 
git push -u origin main

Refresh your GitHub page. Your files are now stored safely online! This is huge! Your code is now global and secure! දැන් ගිහින් බලන්න. ඔබගේ GitHub ගිණුමේ, ඔබගේ සියලුම files සුරක්ෂිතව තැන්පත් වෙලා! මෙය විශිෂ්ටයි! ඔබගේ නිර්මාණ දැන් ලෝක ව්‍යාප්තයි!

Phase 4: The Champion's Launch අදියර 4: ශූරයෙකුගේ මෙවලම්

The Final Mission: Go Live! (Deployment) අවසාන මෙහෙයුම: ලෝකයටම පෙන්වමු (Deployment)

Your project is on GitHub, but it doesn't have a live web address. Let's fix that in 60 seconds using a service called Vercel. අපේ ව්‍යාපෘතිය GitHub එකේ තිබුනට, එයට live web address එකක් නැහැ. Vercel වැනි සේවාවක් භාවිත කර, මිනිත්තුවකින් එය සජීවී කරමු.
